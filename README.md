**QuantumBank Pro** is a cutting-edge, single-page application (SPA) template designed for next-generation fintech platforms, digital banks, and cryptocurrency exchanges. Built purely with HTML, Vanilla JS, and Tailwind CSS, it boasts a stunning dark-mode-first, glassmorphism interface.

[👉 **View Live Demo Here**](crypto-bank-production-8f1f.up.railway.app) 

## ✨ Key Features

* **Deep Dark Glassmorphism UI:** A modern, frosted-glass aesthetic with custom deep blue/purple brand color palettes and smooth transitions.
* **Dynamic SPA Routing:** Built-in Vanilla JavaScript routing to seamlessly switch between Dashboard, Banking, Crypto, and Admin views without page reloads.
* **Interactive Data Visualization:** Integrated with **Chart.js** to render customized, tool-tipped portfolio performance charts.
* **Authentication Flow:** Beautiful, animated modal interfaces for User Login and Registration.
* **API-Ready Architecture:** Includes a structured frontend API interaction layer (`apiCall` wrappers) ready to be hooked up to a real backend (Node.js, Python, etc.).
* **Custom Toast Notifications:** A built-in, animated notification system for success, error, and info alerts.

## 🧩 Dashboard Views Included

* **Dashboard:** Overview of total portfolio value, Fiat vs. Crypto balances, recent activity, and 7-day performance charts.
* **Banking:** Interfaces for Fiat accounts, P2P/Bank transfers, and simulated deposit/withdrawal flows.
* **Crypto Exchange:** Dedicated view for cryptocurrency asset tracking and trading (UI placeholder).
* *(Includes skeleton structures for Wallet, Cards, Loans, Settings, and an Admin Panel)*.

## 🛠️ Technology Stack

* **HTML5**
* **Tailwind CSS** (via CDN for rapid styling)
* **Vanilla JavaScript** (State management, routing, and API logic)
* **Phosphor Icons** (Sleek, modern web iconography)
* **Chart.js** (Data visualization)
* **Moment.js** (Time and date formatting for transactions)
* **Google Fonts** (Inter & JetBrains Mono)

## 🚀 How to Use / Installation

Because this template utilizes CDN links and Vanilla JS, no complex Node modules or build steps are strictly required to view the UI.

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Smaron47/quantumbank-pro-fintech-crypto-dashboard.git](https://github.com/Smaron47/quantumbank-pro-fintech-crypto-dashboard.git)
    ```
2.  **Open locally:** Simply double-click the HTML file to launch the dashboard in your default browser.
3.  **Customize:** The JavaScript contains a `BASE_API_URL` variable (`http://localhost:8080/api`). Update this to connect the frontend to your own live backend!
